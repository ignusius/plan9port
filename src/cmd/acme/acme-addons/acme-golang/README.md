Quick start tutorial
--------------------
For install use:
$./INSTALL

How it use?
-----------
Open acme and open golang source code. Run 'gogo' command.
* gocomp - autocomplete and godoc
* goformat - format code (gofmt)
* gorun  - run code (go run)
